About

This page documents visualizations of the ARRL Field Day 2010-2025 data.

Setup

Here I source the {ggplot2} library to make it easier to make plots later. I also define some default colors for those plots. Defining these here makes it easier to modify them site-wide later on.

Code 
library(ggplot2)

max_color = "red"
median_color = "blue"
tot_color = "green"

Import

The gathering and cleaning pages handle those functions. They save a cleaned data file that we reimport here.

ImportantDRY WIT

A popular adage in computer science and data science is DRY WIT: Don’t Repeat Yourself; Write It Down.

I adopt that mantra here by automating each step and by saving the intermediate products.

Code 
fd <- readr::read_csv("data/aggregate/field-day-2010-2025.csv", show_col_types = FALSE)

Generate summaries

To begin, we create a smaller data summary.

Code 
fd_by_year <- fd |>
  dplyr::group_by(year) |>
  dplyr::summarize(
    median_score = median(final_score, na.rm = TRUE),
    max_score = max(final_score, na.rm = TRUE),
    tot_score = sum(final_score, na.rm = TRUE),
    median_xmtrs = median(transmitters),
    max_xmtrs = max(transmitters),
    median_particip = median(number_of_participants, na.rm = TRUE),
    max_particip = max(number_of_participants, na.rm = TRUE),
    tot_particip = sum(number_of_participants, na.rm = TRUE),
    median_qsos = median(total_qsos, na.rm = TRUE),
    max_qsos = max(total_qsos, na.rm = TRUE),
    tot_qsos = sum(total_qsos, na.rm = TRUE),
    n_stations = length(unique(call))
  )

# Visualize

## About

This page documents visualizations of the ARRL Field Day 2010-2025 data.

## Setup

Here I `source` the {ggplot2} library to make it easier to make plots later. I also define some default colors for those plots. Defining these here makes it easier to modify them site-wide later on.

```{r}
library(ggplot2)

max_color = "red"
median_color = "blue"
tot_color = "green"
```

## Import

The gathering and cleaning pages handle those functions. They save a cleaned data file that we reimport here.

::: {.callout-important}
## DRY WIT

A popular adage in computer science and data science is DRY WIT: Don't Repeat Yourself; Write It Down.

I adopt that mantra here by automating each step and by saving the intermediate products.

:::

```{r}
fd <- readr::read_csv("data/aggregate/field-day-2010-2025.csv", show_col_types = FALSE)
```

## Generate summaries

To begin, we create a smaller data summary.

```{r}
fd_by_year <- fd |>
  dplyr::group_by(year) |>
  dplyr::summarize(
    median_score = median(final_score, na.rm = TRUE),
    max_score = max(final_score, na.rm = TRUE),
    tot_score = sum(final_score, na.rm = TRUE),
    median_xmtrs = median(transmitters),
    max_xmtrs = max(transmitters),
    median_particip = median(number_of_participants, na.rm = TRUE),
    max_particip = max(number_of_participants, na.rm = TRUE),
    tot_particip = sum(number_of_participants, na.rm = TRUE),
    median_qsos = median(total_qsos, na.rm = TRUE),
    max_qsos = max(total_qsos, na.rm = TRUE),
    tot_qsos = sum(total_qsos, na.rm = TRUE),
    n_stations = length(unique(call))
  )
```

## Scores by year

This section visualizes Field Day scores across the 2010-2025 period.

::: {.panel-tabset}

### Median

```{r}
#| label: fig-fd-median-score-2010-2025
#| fig-cap: "Median Field Day score 2010-2025"
fd_by_year |>
  ggplot() +
  geom_point(aes(x = year, y = median_score)) +
  geom_line(aes(x = year, y = median_score), color = median_color) +
  ylab("Median score")
```

### Maximum

```{r}
#| label: fig-fd-max-score-2010-2025
#| fig-cap: "Maximum Field Day score 2010-2025"
fd_by_year |>
  ggplot() +
  geom_point(aes(x = year, y = max_score)) +
  geom_line(aes(x = year, y = max_score), color = max_color) +
  ylab("Max score")
```

### Maximum & median

```{r}
#| label: fig-fd-max-median-scores-2010-2015
#| fig-cap: "Maximum (red) and median (blue) Field Day scores 2010-2025"
fd_by_year |>
  ggplot() +
  geom_line(aes(x = year, y = max_score), color = max_color) +
  geom_line(aes(x = year, y = median_score), color = median_color) +
  ylab("score")
```

### Total

```{r}
#| label: fig-fd-total-score-2010-2025
#| fig-cap: "Total Field Day scores 2010-2025"
fd_by_year |>
  ggplot() +
  geom_line(aes(x = year, y = tot_score), color = tot_color) +
  ylab("Score total")
```

:::
